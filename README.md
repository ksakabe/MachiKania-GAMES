# KM-BASIC 花火シミュレーション

MachiKaniaのKM-BASICで動作する、シンプルな花火シミュレーションです。

花火玉が画面下から上昇し、最高点付近で32個の火花に分裂します。火花は重力の影響を受けながら放物線状に広がり、最後は暗くなって消えます。

## 主な機能

- 花火玉の打ち上げと上昇運動
- 32個の火花によるランダムな爆発
- 重力を使った放物運動
- 火花の軌跡表示
- 寿命に応じた火花の減光
- ランダムな打ち上げ位置と高さ
- 花火の自動連続打ち上げ
- 任意のキーによる終了


## 対応環境

プログラムは起動後に`SYSTEM(22)`と`SYSTEM(23)`を呼び出し、現在のグラフィック画面サイズを取得しています。

``` BASIC
GW=SYSTEM(22)
GH=SYSTEM(23)
```

打ち上げ位置、速度、重力、画面端判定は、この幅と高さを基準に計算しているので様々なサイズのの画面でも、花火の構図が大きく崩れないようにしています。

## 導入方法

1. このリポジトリから`HANABI.BAS`を取得します。
2. SDカードなど、使用するMachiKaniaに対応した方法でファイルを転送します。
3. MachiKaniaのファイル選択画面またはプログラムエディターから`HANABI.BAS`を開きます。
4. 実行前に、機種ごとのプログラム停止方法を確認します。

## 実行方法

### キーボードを接続している場合

1. KM-BASICのエディターで`HANABI.BAS`を開きます。
2. F4キー（RUN）を押します。
3. 花火の連続打ち上げが開始されます。
4. 終了するには任意のキーを押します。

### キーボードを接続していない場合

1. MachiKaniaのファイル選択画面で`HANABI.BAS`を選択します。
2. 使用機種に対応する実行ボタンを押します。
3. 終了操作は機種によって異なるため、MachiKania本体の説明を確認してください。

## プログラムの仕組み

花火の動作は、`PHASE`変数を使った3つの状態に分かれています。

| `PHASE` | 状態     | 処理内容                                 |
|:-------:|----------|------------------------------------------|
|   `0`   | 打ち上げ | 花火玉を上昇させ、重力で速度を低下させる |
|   `1`   | 爆発     | 32個の火花を移動・描画し、徐々に暗くする |
|   `2`   | 待機     | 一定時間待ってから次の花火を打ち上げる   |

### 固定小数点座標

位置は実際の画面座標の8倍で保存しています。

```basic
REM 内部では画面座標の8倍で位置を保持する
X(I)=X(I)+VX(I)

REM 描画時に8で割って画面座標へ戻す
NEWX=X(I)/8
```

これにより、浮動小数点を使わずに1ドット未満の細かな移動を表現しています。

### 重力

火花の縦方向速度に毎フレーム一定値を加えます。

```basic
REM 正の方向は画面下側。縦解像度に合わせた重力を加える
VY(I)=VY(I)+(2*GH+112)/224
```

上向きに飛び出した火花も、時間が経つと上昇速度を失い、下向きに落下します。

### 火花の減光

火花の寿命が残り16フレーム未満になると、暗い色へ切り替えます。

```basic
REM 寿命が短くなったら対応する暗色パレットを使う
COL=C(I)
IF LIFE<16 THEN COL=COL+8
```

### 画面サイズへの追従

ロケットと火花は、横方向を`GW/256`、縦方向を`GH/224`の比率で調整します。

## カスタマイズ

### 火花の数を変更する

現在は0～31番の32個です。たとえば48個にする場合は、関連する配列とループをすべて0～47へ変更します。

```basic
REM 48個分の配列を確保する
DIM X(47)
DIM Y(47)
REM 2文字の配列名は事前にUSEVARで宣言する
USEVAR VX
USEVAR VY
DIM VX(47)
DIM VY(47)
DIM C(47)

REM 48個の火花を処理する
FOR I=0 TO 47
```

配列サイズとすべての処理ループを同じ数にしてください。一部だけ変更すると、配列範囲外へのアクセスや不正動作の原因になります。

### 火花の表示時間を変更する

```basic
REM 値を大きくすると火花が長く残る
LIFE=48
```

### 重力を変更する

```basic
REM 先頭の2を小さくすると、火花がゆっくり落下する
VY(I)=VY(I)+(2*GH+112)/224
```

### 打ち上げ速度を変更する

```basic
REM 68と13を大きくすると高く上がる
RVY=(-68-(RND()%13))*GH/224
```

画面では上方向が負なので、負の値を大きくすると高く上昇します。

### 描画速度を変更する

```basic
REM 2は約2/60秒の待ち時間
WAIT 2
```

値を小さくすると速く、大きくすると遅くなります。ただし、実際の速度は機種と描画負荷にも左右されます。

### 色を変更する

```basic
REM 書式: GPALETTE 色番号,赤,緑,青
GPALETTE 2,255,70,40
```

赤・緑・青の値はそれぞれ0～255です。色番号1～7が明色、9～15が対応する暗色として使われています。
